# 🏢 Facility Administration Setup Guide

## Overview
The Elder Assistant now includes a dedicated admin interface for facility staff to manage community events and monitor resident activities.

## Admin Access Credentials

### Default Admin Accounts
1. **Facility Administrator**
   - Username: `admin`
   - Password: `facility2024`
   - Role: Full administrative access

2. **Staff Member**
   - Username: `staff`  
   - Password: `staff123`
   - Role: Limited staff access

## How to Access Admin Interface

1. **Launch the Elder Assistant application**
2. **Navigate to the "🏢 Admin" tab** (far right)
3. **Login with admin credentials** using the form provided
4. **Access admin features** after successful authentication

## Admin Features

### 📅 Manage Events
- **Create new community events** with details like:
  - Event name and description
  - Day of week and time
  - Location within facility
  - Recurrence pattern (weekly/monthly/one-time)
- **View all active events** in expandable cards
- **Edit/Deactivate events** (coming soon)

### 👥 Resident Info
- **View system usage statistics**:
  - Total residents in system
  - Daily active users
  - Voice vs text interaction counts
  - Emergency alert tracking
- **Monitor engagement levels**

### 📊 Reports & Analytics
- **Usage trend charts** showing daily activity
- **Event participation statistics**
- **Generate comprehensive reports** (feature coming soon)

## Event Creation Process

1. **Go to Admin tab → Manage Events**
2. **Fill out the "Create New Event" form:**
   - Enter event name (e.g., "Yoga Class")
   - Add description (e.g., "Gentle yoga for all skill levels")
   - Select day of week
   - Choose time
   - Specify location (e.g., "Recreation Room")
   - Set recurrence pattern
3. **Click "🎉 Create Event"**
4. **Event appears immediately** in resident Community tab

## Integration with Resident Interface

- **Events created by admin** automatically appear in the resident Community tab
- **Residents can view upcoming events** in their social feed
- **System tracks participation** and usage analytics
- **No interference** between admin and resident interfaces

## Security Features

- **Separate authentication** system for admin access
- **Role-based permissions** (admin vs staff levels)
- **Session management** with logout functionality
- **Credential storage** in secure JSON format

## For Facility Staff Training

### Daily Operations:
1. **Check usage statistics** in Resident Info tab
2. **Review upcoming events** in Manage Events
3. **Add new activities** as needed using event creation form
4. **Monitor emergency alerts** (if any occur)

### Weekly Tasks:
1. **Plan next week's activities** using event scheduler
2. **Review participation reports** in analytics
3. **Update recurring events** if schedules change
